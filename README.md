# FH6 Garage Companion

A personal garage manager and race companion for Forza Horizon 6.

Built because the in-game garage is great for owning cars and terrible for deciding which one to actually use.

## What it does

- **Car database** — full FH6 car list, sortable and filterable by class, PI, division, and country
- **Your garage** — add cars you own, view them in list or grid, sort and filter the same way
- **Tags** — tag your garage cars by driving style, surface, and condition so you can find the right car fast
- **Race types** — reference guide for each race discipline covering what it demands, what to avoid, and what PI range is competitive
- **Race tray** — open a race type while browsing your garage and the list filters to your matching cars instantly

## Stack

- Next.js + Tailwind CSS
- Prisma + PostgreSQL
- Docker (local development)

## Status

Active personal project. Features being added as the game gets played.

## Roadmap

- Auto-tagging cars by division and drivetrain
- Car spec data (HP, torque, weight, drivetrain) when community sources become available
- Tuning suggestions based on car type and race
- Build sandbox for planning upgrades before getting in game
